Projet académique pour le cours de deep learnig.

Classification sur le jeu de données MNIST avec seulement 100 exemples étiquetés et 59 900 exemples sans labels pour l'entraînement. 
Implémentation d'une méthode d'apprentissage semi-supervisé basée sur l'article scientifique de GILDARIS et comparaison avec un modèle basique.
